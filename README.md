# Unit Converter

- This application is can be used for converting values from one measuring system to another.
- The 3 unit conversions that happen are
	- Liters to Gallons
	- Meters to feet
	- Kilos to Pounds

### Tech Used

- JavaScript
- CSS
- HTML
